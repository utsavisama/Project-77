What is visual studio code ?

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git

How to set up Visual Studio code ?

1.Download and install VS Code. 2.Create a new file. 3.See an overview of the user interface. 4.Install support for your favorite programming language. 5.Change your keyboard shortcuts and easily migrate from other editors using keybinding extensions. 6.Customize your editor with themes. 7.Explore VS Code features in the Interactive Editor Playground

Why is a console screen used ??

Console screen is mainly used to check our code.

How to open a console screen ??

To open a console screen first we have to right click any where at any webpage and then select inspect, After that we have to click at console option and we can check our code.
